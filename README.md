## Quick Start

1. Clone this repository:
   ```bash
   git clone https://github.com/yourusername/your-repo.git
   ```

2. Open `index.html` in your browser

3. Customize content:
   - Edit company name and contact info in `index.html`
   - Modify colors by changing CSS variables in `styles.css`
   - Update projects and services to match your business

## Deploying to GitHub Pages

1. Push your code to GitHub
2. Go to your repository Settings ‚Üí Pages
3. Select "Deploy from a branch"
4. Choose `main` branch and `/root` folder
5. Your site will be live at: `https://yourusername.github.io/your-repo/`

## Customization Guide

### Change Colors
Open `styles.css` and modify the CSS variables at the top:
```css
:root {
    --primary-color: #667eea;
    --secondary-color: #764ba2;
    --accent-color: #f5576c;
}
```

### Update Company Information
- Edit company name in navbar logo
- Update contact details in the contact section
- Modify business description in the about section
- Add your social media links in footer

### Add Your Projects
Add new project cards in the projects section with your actual project details and images.

### Update Services
Customize the services section to match your offerings.
